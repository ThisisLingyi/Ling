# Lingyi Ge — Personal Website

Static portfolio site. No build step — open `index.html` (or `Home.dc.html`) in a browser.

## Pages
- `Home.dc.html` — homepage
- `Case-AI-Onboarding.dc.html` — case study: AI & Guided Onboarding
- `Case-JTBD.dc.html` — case study: JTBD in Data Management

## Assets
- `support.js` — runtime that renders the pages
- `uploads/` — headshot and résumé PDF

## Deploy (GitHub Pages)
Push this folder to a repo, then enable Pages on the default branch (root). `index.html` redirects to the homepage.
